## Project Overview

This project analyzes India’s economic growth across different political regimes using Python and data analytics techniques. The analysis focuses on key economic indicators such as GDP growth, inflation, unemployment, and economic trends during various prime ministerial periods. The goal is to derive data-driven insights into how political leadership influenced India’s economic performance over time.

## Objectives
Analyze economic growth trends across government tenures                                                                                                
Compare major economic indicators between political periods                                                                                                     
Identify patterns, fluctuations, and growth phases                                                                                                      
Visualize insights using charts and dashboards
